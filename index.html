Subir proyectos de Angular a Firebase

Paso 1: Crear un proyecto en Firebase

Ve a la consola de Firebase (https://console.firebase.google.com/) y crea un nuevo proyecto.

Paso 2: Configurar el proyecto de Angular

Abre tu proyecto de Angular y ejecuta el siguiente comando para instalar Firebase CLI:

npm install -g firebase-tools

Luego, autentícate con Firebase usando el siguiente comando:

firebase login

Después de la autenticación, vincula tu proyecto de Angular a Firebase usando:

firebase init

Elige "Hosting" como función y selecciona tu proyecto de Firebase.

Paso 3: Construir el proyecto de Angular

Genera una versión optimizada de tu proyecto de Angular usando:

ng build --prod

Esto creará una carpeta "dist" en tu proyecto.

Paso 4: Implementar el proyecto en Firebase

Finalmente, implementa tu proyecto en Firebase usando:

firebase deploy

Después de que se complete la implementación, recibirás una URL donde puedes acceder a tu proyecto.

Objetivo

El objetivo de este ejercicio es adquirir los siguientes conocimientos.

  • 1. Uso de componentes en Angular
  • 2. Despliegue del proyecto Angular en Firebase
  • Documentación de Angular
  • Página de Firebase
